## triangle.py
-**area(a, h)**- Функция принимает два знаечния: a - длина основания треугольника, h - длина высоты треугольнка, проведенной к а. Возвращает площадь треугольника, вычисленную по формуле (a * h)/2.
Пример работы кода:
    a = 10;
    h = 5; 
    area(a,h) = area(5,10) = 5*10/2=25

-**perimeter(a, b, c)**- Функция принимает 3 параметра: a, b, c, являющиеся длинами сторон треугольника. Вычисляет значение периметра по формуле a + b + c. Возвращает значение периметра, вычисленное по формуле a + b + c.
Пример работы кода:
    a = 10;
    b = 15;
    c = 20;
    perimetr(a,b,c) = perimetr(10,15,20) = 45

## square.py
-**area(a)**- Функция принимает числовое значение а - длина стороны квадрата. Возвращает значение площади квадрата, вычисленное по формуле a * a.
Пример работы кода:
    a = 10;
    area(a) = area(10) = 10 *10 = 100

-**perimeter(a)**- Функция принимает числовое значение а - длина стороны квадрата. Возвращает значение периметра квадрата, вычисленное по формуле 4*а.
Пример работы кода:
    a = 10;
    perimetr(a) = perimetr(10) = 4 *10 = 40

## rectangle.py
-**area(a, b)**- Функция принимает два параметра a и b, которые являются длинами перпендикулярных сторон прямоугольника. Возвращает площадь прямоугольника, вычисленную по формуле a*b.
Пример работы кода:
    a = 10;
    b = 5; 
    area(a,b) = area(10,5) = 10 *5 = 50

-**perimeter(a, b)**- Функция принимает два параметра a и b, которые являются длинами перпендикулярных сторон прямоугольника. Возвращает периметр прямоугольника, вычисленный по формуле 2*a+2*b.
Пример работы кода:
    a = 10;
    b = 5; 
    perimetr(a,b) = perimetr(10,5) = 2 *10 + 2 *5 = 30

## circle.py
import math Импорт math для использования pi.


-**area(r)**- Функция принимает один параметр: r - радиус окружности. Возвращает площадь круга, вычесленную по формуле math.pi * r * r.
Пример работы кода:
    r = 10;
    area(r) = area(10) = math.pi*100 = 314.1592653589793
-**perimeter(r)**- Функция принимает один параметр: r - радиус окружности. Возвращает длину окружности, вычисленную по формуле 2 * math.pi * r
Пример работы кода:
    r = 10;
    perimetr(r) = perimetr(10) = 2 * math.pi * 10 = 62.83185307179586
